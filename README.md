# OutBid

Real-time competitive bidding game.

## Structure

\`\`\`
OutBid/
├── frontend/   # Next.js App Router + TypeScript
└── backend/    # Fastify + Socket.io + Drizzle
\`\`\`

## Quick start

**Backend**
\`\`\`bash
cd backend
cp .env.example .env   # fill in your DB + Redis URLs
npm run dev
\`\`\`

**Frontend**
\`\`\`bash
cd frontend
npm run dev
\`\`\`
